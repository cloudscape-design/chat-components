React component library for conversation-oriented UI in the Cloudscape Design System.

## Commands

```sh
npm start          # Dev server with file watching
npm run build      # Production build
npm run lint       # ESLint + Stylelint
npm run test:unit  # Unit tests (vitest)
npm test           # Lint + unit tests
```

## Conventions

This repo follows the same conventions as the [main Cloudscape components library](https://github.com/cloudscape-design/components). See the detailed guides (covers component structure, props, events, styling, writing testing, code style, dev pages, and API docs): https://github.com/cloudscape-design/components/blob/dev-v3-amanabiy/docs/CLOUDSCAPE_COMPONENTS_GUIDE.md
