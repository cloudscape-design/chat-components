# Chat Components

React components for conversation-oriented UI in the Cloudscape Design System.

## Quick Reference

- Build: `npm run build`
- Lint: `npm run lint`
- Test: `npm test`

## Development Conventions

This repo follows the same conventions as the main components library. For component structure, props, styling, testing, and accessibility guidelines, see the [components repo docs](https://raw.githubusercontent.com/cloudscape-design/components/main/docs/).
